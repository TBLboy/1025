# 记忆向量库备份

这个文件夹用于备份向量记忆库的数据到 GitHub 远程仓库。

## 目录结构

```
backup/
├── archive/              # JSON 格式的记忆备份
│   └── 2026/
│       └── 03/
│           └── 2026-03-24.json
├── daily_summary/        # 每日摘要
│   └── 2026-03-24.md
└── manifest.json         # 备份清单
```

## 备份策略

- **实时备份**: 每条消息存储时同步更新
- **每日归档**: 每天凌晨生成完整备份
- **版本控制**: 通过 Git 提交历史追踪变更

## 恢复方法

```bash
# 从远程仓库拉取最新备份
git pull origin main

# 恢复指定日期的备份
cp backup/archive/2026/03/2026-03-24.json ../archive/2026/03/
```
